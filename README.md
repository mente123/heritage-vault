# Heritage Vault — African objects at the British Museum

A map of **all 54 African countries** showing objects from the British Museum’s collection (25,000+ artefacts in this dataset). Data is grouped by country; click a country to see its artefacts and links to the collection.

**Run locally:** `npx serve` then open http://localhost:3000  
**Live:** [mente123.github.io/heritage-vault](https://mente123.github.io/heritage-vault)

## Data source

- [British Museum — Collection search (Africa)](https://www.britishmuseum.org/collection/search?place=Africa)
- [British Museum — Contested objects](https://www.britishmuseum.org/about-us/british-museum-story/contested-objects-collection)
